# cadena-de-frio
***
descripcion

El proyecto consiste en el desarrollo de una plataforma web integrada con tecnologías IoT (Internet de las Cosas) e Inteligencia Artificial para el monitoreo, automatización y control en tiempo real de la cadena de frío en contenedores o vehículos de transporte de productos médicos críticos (como vacunas y medicamentos sensibles).
El sistema físico consta de un módulo embebido equipado con sensores que miden constantemente variables ambientales y de seguridad del contenedor (temperatura, humedad y estado de la puerta). Estos datos se transmiten de forma instantánea a una base de datos distribuida y a un backend con lógica predictiva basada en Inteligencia Artificial. La plataforma web centraliza esta telemetría para que el personal de logística y los operadores puedan auditar las condiciones del envío mediante dashboards interactivos, recibir alertas críticas ante pérdidas de frío y consultar un asistente inteligente para la toma de decisiones rápidas ante contingencias en la ruta.
